## Questions They'll Ask

**"How does the rent estimate work?"**
We use RentCast API for market-level data, but your PM network overrides it. A PM's actual vacancy rate is worth more than any API estimate.

**"What does it cost investors?"**
Beta is free. Long-term model is referral-based — you bring a client to close, we work out a fair referral structure together.

**"Who built this?"**
Built from scratch around our own deals. Pythias and Roanoke are the first two properties in the system. Real numbers, real analysis.

**"Can I add my own properties to the network?"**
That's the V1 network portal — you submit a deal with your data, it hits the right investor dashboards automatically.

**"Is this AI-generated garbage?"**
The financial model is fully transparent — every number is auditable. The AI writes the narrative summary, but the numbers drive everything. AI can't override the math.

## Contacts to Onboard First

For the beta, you need at least one of each:
- [ ] Agent (deal submission + verification)
- [ ] Property manager (vacancy data + CMI input)
- [ ] Lender (rate quotes + pre-approval flow)
- [ ] Contractor (rehab estimates for BRRRR/rehab deals)

Start with whoever you trust most. The network quality IS the product.
